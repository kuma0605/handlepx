# divide css value with unit into half 

## install
npm i pxtohalf -g

## usage          
pxtohalf go [folderPath] [cssFileExtension]

## Note:
1px will not be converted.

### arguments
#### folderPath 
optional, default: __dirname

#### cssFileExtension
optional, default: css

### exmaple
pxtohalf go ./css scss
